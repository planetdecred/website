Chat

Web interface and Sign Up:

Use chat.planetdecred.org. For development, please use the same name you use on github.
Once logged in, join the Planet Decred room by clicking on Explore on the top left.

Mobile apps:

Android chat and iOS chat can be configured with the matrix.planetdecred.org home server.

Matrix users:

If you already have a matrix account, get a list of our rooms in your matrix client by selecting
Explore → Matrix Rooms → Add a new server → planetdecred.org
